## 2. 이익 추종 전략 (Let Profits Run)

### 전략 설명
수익이 커질수록 트레일링을 강화하여 큰 수익을 끝까지 추종합니다.

| 수익률 | 트레일링 | 손절 |
|--------|----------|------|
| +8% 도달 | 5% 활성화 | 본전으로 이동 |
| +15% 도달 | 3%로 강화 | - |
| +25% 도달 | 2%로 더 강화 | - |

**고정 익절 없음** - 무제한 상승 추종

### 결과
```
총 수익률:        +261.66%
연평균 수익률:     51.70%
MDD:              -9.88%
샤프 비율:         2.56
```

### 청산 사유별 통계
| 청산 사유 | 횟수 | 평균 수익률 |
|----------|------|------------|
| 보유기간 만료 | 551회 | +3.53% |
| 손절 | 180회 | -7.00% |
| 트레일링 L1 (+8%~15%) | 48회 | +4.90% |
| 트레일링 L2 (+15%~25%) | 38회 | +15.25% |
| 트레일링 L3 (+25%↑) | 13회 | **+36.58%** |

### 핵심 성과
- **대박 종목 포착**: L3 트레일링 13회, 평균 **+36.58%** 수익
- **MDD 감소**: 11.26% → 9.88% (위험 감소)
- **평균 수익 증가**: 5.93% → 8.86%

## 5. 결론 및 권장사항

### 이익 추종 전략 채택 이유
1. **수익률 2배**: 135% → 261%
2. **위험 감소**: MDD 11.26% → 9.88%
3. **효율성 향상**: 적은 거래로 더 높은 수익
4. **대박 종목 포착**: +25% 이상 종목에서 평균 +36% 수익

### 실전 적용 시 주의사항
1. 백테스트는 과거 데이터 기반이므로 미래 수익을 보장하지 않음
2. 실전에서는 슬리피지, 호가 스프레드 등 추가 비용 발생
3. 시장 상황에 따라 전략 파라미터 조정 필요
4. 최소 3개월 모의투자 후 실전 적용 권장
